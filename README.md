# C++ Pattern Programs

This repository contains a collection of basic to advanced pattern printing programs implemented in C++. The programs are designed to strengthen logic building, nested loops, and problem-solving skills for beginners learning C++.

## Topics Covered

- Star Patterns
- Number Patterns
- Pyramid Patterns
- Inverted Patterns
- Hollow Patterns
- Diamond Patterns
- Symmetric Patterns
- Character Patterns

## Tech Stack

- Language: C++
- Compiler: g++
- IDE: Visual Studio Code

## File Structure

```text
p01.cpp
p02.cpp
p03.cpp
...
```

Each file contains a separate pattern program.

## Purpose

This repository was created for:

- Practicing C++ fundamentals
- Improving loop logic
- Strengthening DSA basics
- Preparing for coding interviews and problem solving

## How to Run

Compile and run any file using:

```bash
g++ filename.cpp -o output
./output
```

Example:

```bash
g++ p01.cpp -o p01
./p01
```
